The objective of this project is to analyze customer shopping behavior and generate meaningful business insights through data analysis and visualization.

The project follows a complete analytics pipeline:

- Load dataset using Python
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess data
- Store cleaned data in MySQL
- Execute SQL queries for business insights
- Build an interactive Power BI dashboard
- Create a project report
- Prepare a presentation using Gamma AI

Clone this repository.

```bash
git clone https://github.com/yourusername/customer-shopping-analysis.git
```

Install the required Python libraries.

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy pymysql
```

### Step 3

Open **Data.ipynb** in Jupyter Notebook.

### Step 4

Run all notebook cells to:

- Load the dataset
- Perform EDA
- Clean the data
- Export data to MySQL

### Step 5

Open the SQL file in MySQL Workbench and execute the queries.

Open the **Customer Behaviour.pbix** file in Power BI Desktop to explore the interactive dashboard.
